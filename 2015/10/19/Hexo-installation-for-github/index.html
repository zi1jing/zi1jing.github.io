Hexo installation for github

Hexo 搭建github静态网站

  • 安装
    1. 安装git
    2. 安装node.js

  • 配置

    1. 在github上建立repository。

      说明：仓库名必须为“ username.github.io ”

    2. 配置ssh-key

      生成密钥：

      按3个回车，密码为空。
      最后得到了两个文件：id_rsa和id_rsa.pub（公钥）
      用username登录github网站，找到settings, 并添加密钥

  • 安装hexo

    1. 打开git bash命令行，定位到博客文件夹下

    2. 打开服务

      表明Hexo Server已经启动了，在浏览器中打开 http://localhost:4000/，这时可以看到Hexo已为你生成了一篇blog。

    3. 配置_config.yml文件

  • 写博客

    1. 生成新博客

      根据上一步中给出的路径，用记事本等编辑软件打开并编辑博客

    2. 本地调试

      hexo s -g

      hexo clean
      hexo generage (or “hexo g”)
      hexo deploy (or “hexo d”)
      ```
